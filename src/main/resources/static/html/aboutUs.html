Sobre Pattsberry

La marca Pattsberry surge por el gusto y pasión por crear cosas de una forma artesanal ¡Cada producto puede ser personalizado! Si tienes algún proyecto especial podemos ayudarte a llevarlo a la realidad.

Amigurumis

Cada uno de nuestros amigurumis es tejido especialmente para ti, utilizando el mejor material para elaborarlos, como resultado tendrás un muñequito único.

Libretas

Nuestras libretas son hechas a mano y personalizadas según tus necesidades, manejamos diferentes formatos como carta, media carta y cuarto de carta, personalizamos tanto las hojas como las portadas.
